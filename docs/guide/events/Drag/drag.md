## 事件名称

| 事件名  | 说明 |
| --- | --- |
| `dragStart` | 开始拖动事件 |
| `drag` | 拖动事件中 |
| `dragEnd` | 结束拖动事件 |
| `dragOver` | 拖动元素 over |
| `dragOut` | 拖动元素 out |
| `dragEnter` | 拖动元素进入 |
| `dragLeave` | 拖动元素离开 |
