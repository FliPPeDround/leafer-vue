## 事件名称

| 事件名  | 说明 |
| --- | --- |
| `swipeLeft` | 向左滑动事件 |
| `swipeRight` | 向右滑动事件 |
| `swipeUp` | 向上滑动事件 |
| `swipeDown` | 向下滑动事件 |
